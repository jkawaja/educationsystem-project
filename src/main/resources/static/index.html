Welcome to my project page. Use the 'RunScript' Class to populate the database.

You can use the following APIs for testing in Postman:

  1. http://localhost:8080/educationsystem-project/api/cna/course/add
  2. http://localhost:8080/educationsystem-project/api/cna/course/list
  3. http://localhost:8080/educationsystem-project/api/cna/course/view/{courseID}
  4. http://localhost:8080/educationsystem-project/api/cna/course/delete/{courseID}
